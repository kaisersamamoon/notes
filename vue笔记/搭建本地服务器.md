## 搭建本地服务器

1. 第一步安装模块

   ```
   npm install webpack-dev-server@2.9.1 --save-dev
   ```

   2. 新版本配置文件

```
"dev": "webpack serve --mode development --env development --open chrome"
```



## webpack  配置文件分离

production 生产模式
